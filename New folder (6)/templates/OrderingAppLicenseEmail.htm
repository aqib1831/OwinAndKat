Pre-Installation Requirements
  • You will need to be running version SMSx version 20130701a or above. You can easily check this by looking at the top right corner of your SMSx system. If you're running version 415a or lower please contact Netsolace technical support for assistance upgrading.
  • WIFI enablement at your store via the same router your SMSx system is using to connect to the internet
  • iPad 2 or above running a version of iOS 6 available to install and run the Fruit Expert app.
  • Ordering App web service MUST be enabled by NS technical support
