New Additions   Print Notes
 
1 - Qs Training Mode
 

Qs can now be run in two modes:

    1. Training
    2. Live

Training mode within Netsolace Qs has been made available for proper training of employees before store openings. This mode will not report sales on Socket Fransupport which is the main difference between both modes.

This selection of mode can be performed from the Netsolace Qs startup screen each time until store chooses to select “Live” mode. Once the Live mode is activated then Netsolace Qs will not prompt for mode selection afterwards. and the Live mode cannot be reverted back to Training Mode.
 
2 - Price Restore point
 

Whenever a price change will occur either manually or by synchronization, Netsolace Qs will automatically take a backup for products’ old prices and create a restore point. These backup prices can be accessed from “Pricing” screen under manager menu by clicking on “Restore Prices” button.
Where store manager can preview the old prices, restore the backup prices and can delete any backup prices.
 
3 - History View
 

Whenever a store will change the price of a product, Netsolace Qs will log the price changes and they can be viewed from “pricing” screen under Manager Menu by clicking “View History” button. Store manager can view the list of all price change logs and can view the details of each log by clicking on “view detail” button.
 
4 - Auto Closeout
 

A new process “Auto closeout” has been added in Netsolace Qs. If a dayend is not performed until 11:50 for any day, then Netsolace Qs will automatically perform the dayend between 11:50pm - 12.00am. For auto Dayend only the Payment Gateway application should be running on the store server. If for any reason, Netsolace Qs has been shut down during that time, Auto closeout will not be performed.
 
5 - Add Single Product Automatically

 

If a category has a single product/size then it will be automatically added to the order by just clicking on the category.

 
 
6 - New Reports

  • A new report “payroll format for employee timing” has been added which contain information related to the employees, such as “Social Security #”, “Employee ID”, Earning Type”, “Hours” and “Rate Amount”. This report will show the calculated due wage for an employee according to the hours he/she has worked.
  • A new report “Store sales report” has been added .It shows store sales within the given date ranges.
  • Another report named “Coupon Analysis report” has been added.
  • “Online Transaction report” has been added which shows Online CC, GC transactions detail.
  • “Employee labor cost by hour report” has been added. That contains a graph that shows the employee cost by hours.

 

Enhancement in Existing Features
 
1 -Qs Synchronization Changes
 
  • Synchronization can no longer be run on multiple terminals at the same time.
  • Synchronization criterion has been changed. Corporate office will control the product synchronization for stores.
    If “Auto Synchronize” option is enabled on socket Fransupport and an admin will change the price of a product, then Netsolace Qs will show an alert on home page.
  • Stores will not be synchronized automatically as with previous updates.
 
2 - Arrange Menu
 

A new drop down” Arrange Menu” has been added on “Other settings” screen under store information in Managers menu. This will be used to sort the menu by popularity, popularity – Last 30 days and alphabetically on the “order now” screen.
 
 
3 - Menu Setting

On Menu setting screen a new column has been added to differentiate store defined and online synchronized items.
 
4 - Employee Center

Two new columns have been added in employees list on employee center to show employee’s Home Phone and cell no.
 
5 - Changes in Reports
 
  • A total of all records of a column have been added in all required reports to show “totals” of all columns where ever there is a sum of all records needed.
  • A new column “labor Cost” has been added in following reports
    • Sales amount by hour report
    • Weekly sales report
    • Employee time report
  • A change has been made in Sales by Category report. Now it will show products, options and sizes of products respectively.
  • A change has been made in “Closeout report”.  Now it will show the adjustments made in orders.
 
6- Managing “Shutdown Qs” and “Text exemption” option
 

Now Store Manager can manage access of “Shutdown Qs” and “Text Exemption” features for Operators.  By default, store operators do not have access to both of these features.
